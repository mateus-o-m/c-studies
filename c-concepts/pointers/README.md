Pointer studies in C language



Pointer is a variable that stores the memory address of another variable

Pointer is defined using "\*": int\* agePtr = \&age ===> agePtr is a pointer to an int called age

"\&" operator: is a reference operator, that access the memory address of a variable

"\*" operator: is a dereference operator, that access the value stored at a memory address
